Passive Health Regen 1.3.0 is a big parity and polish update across every supported version.

The core regen features and config options are now lined up across the full version set. That includes the newer saturation settings, campfire support where it belongs, poison and wither healing rules, reconnect cooldown persistence, and the broader bonus system.

The heart HUD also got a full spread across the lineup. The 1.20.1 and 1.21.1 builds use the full modern HUD with smoother fades, cleaner pulse and flash timing, and extra visual states for hunger block, saturation, campfire, poison, wither, and freezing where supported. Older versions keep the same general feedback style, with version-appropriate compromises where the rendering path is rougher.

Fresh configs now default to the timer being off. Saturation bonuses are on by default. Campfire and freezing defaults were also cleaned up for the versions that support them.

This update also fixes a batch of HUD sync and display issues that turned up during the cross-version port pass.
